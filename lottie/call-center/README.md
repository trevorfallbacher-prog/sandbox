# Call Center Lottie Concepts ("Savera"-inspired vibe)

These are lightweight, editable starter Lottie JSON files intended for a modern customer-support/call-center theme:

- `incoming-call-ring.json` — phone-like circular ping/ring pulse.
- `agent-headset-pulse.json` — agent/avatar circle with status pulse.
- `support-chat-wave.json` — chat bubble with subtle wave indicators.

## Notes

- All animations are loop-friendly.
- Palette uses cool blue + cyan accents for a clean, product-style look.
- Designed as concept starters that motion designers can refine in After Effects + Bodymovin.

## Quick preview options

Use any Lottie player (web/mobile) and load the JSON directly.
